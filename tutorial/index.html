First steps

This template comes with everything you need to build an extension. You can find all of the extension code in the code tab, under src/main/HelloJavaExtension.java.

To build your extension:

  1. Either press this button, or type package and hit enter in the search bar on the top:
  2. Go to the extensions folder. You can get their quickly by going to the code tab, hitting the "..." next to Browse Code, and clicking Reveal in system.
  3. In the extension folder, you should see hello-java.zip (or something else if you've changed the extension name). Unzip it to get a hello-java folder that contains hello-java.jar.
  4. Drag the hello-java folder to your NetLogo extensions folder.
  5. Startup NetLogo, put extensions [ hello-java ] in your code tab. Run hello-java:hello "Your Name".

Try to implement these!

Goodbye!

Make a goodbye primitive, just like the hello.

Current time in milliseconds

Try to implement a reporter that returns the current time in milliseconds. You can use System.currentTimeMillis() to get the current time in milliseconds. To return a number from a primitive, make sure you cast it as a Double.

String repeater

Try to implement a reporter that takes a string and a number and returns that string repeated that number of times. For instance, hello-scala:repeat-string "hello" 3 would return "hellohellohello". You can access the first argument to a primitive with args[0] and the second argument to a primitive with args[1].

List joiner

Try to implement a reporter that takes a list and string and returns the items of the list, concatenated together, separated by the string. For instance, hello-scala:join ["one" "two" "three"] "," should return "one,two,three".

To get a list from an argument, use args[[0].getList. This gives you a LogoList. Then, to join the elements of the list together, turn the LogoList into a List, iterate over it, and append each String from inside it.

String splitter

Create a reporter that takes two strings, and splits the first string around occurrences of the second string. For instance, hello-scala:split "one,two,three" "," should return ["one" "two" "three"].

You can split a string like this in Java by doing str.split(delim), where str is the variable containing the string you want to split, and delim contains the string you want to split around. This will give you a String[] array. To return this, you must first convert it to a LogoList. To do this, you can do LogoList.fromJava(Arrays.asList(array)), where array is your array of strings.

Move left and right

Create two commands that tell turtles to move a number units sideways: one for left and one for right. You can base these commands off of Wiggle. To have a turtle move forward, use turtle.jump(x), where x is the distance you want them to move forward.
